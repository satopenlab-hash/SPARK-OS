# OpenLab OS Skills

Skills are modular operating instructions. Use only the skill relevant to the current task.

## How to use

1. Read `CLAUDE.md`.
2. Identify the OpenLab layer and task type.
3. Load the relevant skill.
4. Follow its decision checks.
5. Report confidence, evidence, assumptions, missing evidence, and required human review.
