# Example: Reflection Coach Agent

**Status:** Synthetic demonstration  
**Related layer:** Learning  
**Human approval required:** Yes

## Purpose

Help a learner make their thinking visible after a project activity.

## Inputs

- Learner's draft reflection.
- Teacher-approved reflection prompt.
- Optional evidence links.

## Outputs

- 3–5 clarifying questions.
- A list of evidence the learner has mentioned.
- A list of possible missing evidence.
- Confidence and uncertainty statement.

## Authority

The agent may recommend questions and organization.

## Blocked actions

The agent may not:
- write a final reflection as if it were the learner;
- judge academic performance;
- issue a credential;
- access unapproved student records;
- remove learner voice.

## Stop condition

Stop after one question set and one structured evidence check. Wait for learner or teacher input before continuing.
